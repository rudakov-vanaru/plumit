Задача
Первоначальная задача была разработать Frontend при помощи HTML, CSS, JS по уже готовому дизайну и исправить ошибки других разработчиков.
Исполнение
Разработка Frontend осуществлялось при помощи языка разметки HTML, CSS и языка программирования JavaScript без использования сторонних библиотек.

Была разработана Frontend часть для интернет-магазина Bookingrent и исправлены некоторые ошибки других разработчиков. Был реализован адаптив под мобильные устройсва и была создана "Темная тема".
